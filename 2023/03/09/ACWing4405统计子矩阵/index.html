ACWing 4405. 统计子矩阵

发表于2023-03-09|更新于2023-04-04|算法练习ACWing
|字数总计:790|阅读时长:3分钟|阅读量:

题目描述

给定一个 N×M 的矩阵 A，请你统计有多少个子矩阵 (最小 1×1，最大 N×M) 满足子矩阵中所有数的和不超过给定的整数 K?

输入格式
第一行包含三个整数 N,M 和 K。

之后 N 行每行包含 M 个整数，代表矩阵 A。

输出格式
一个整数代表答案。

数据范围
对于$30%$的数据，$N,M≤20$，
对于$70%$的数据，$N,M≤100$，
对于$100%$的数据，$1≤N,M≤500;0≤A_{ij}≤1000;1≤K≤2.5×10^8$。

输入样例：


输出样例：

样例解释

题解

二维前缀和

二维前缀和预处理矩阵
枚举子矩阵的左上、右下两个端点(x1,x2,x3,x4), 计算子矩阵的和
时间复杂度O(n^4)，需优化
结果： Time Limit Exceeded 通过了 4/10个数据

优化：一维前缀和 + 双指针

视频

优化的方法是：
1) 通过指针 s 和 t 确定矩阵上下边界
2) 一维前缀和处理上下边界内的每一列
3) 枚举子矩阵的 左边界i 和 右边界j，i固定 j向右移动直到子矩阵刚好过大
4) i向右移动，使得子矩阵减小至范围内
5) 每一次竖指针发生变化 ans += 右-左+1
